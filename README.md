# Assignment-8

**Configuration and Setup**

In order to run this project locally, simply fork and clone the repository or download as zip and unzip on your machine.

1. Open the project in your prefered code editor.
2. Go to terminal of the code editor :
    ##### Change directory to assignment8
    ```python
    cd assignment8
    ```
    ##### Install dependencies
    ```python
    npm install
    ```
    ##### Start the development server
    ```python
     npm run dev
    ```
3. Go in browser : 
   ```python
    http://localhost:5173/
   ```
